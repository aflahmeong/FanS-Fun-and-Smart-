# FanS-Fun-and-Smart-
FanS merupakan salah satu media pembelajaran anak berumur 3-5 tahun yang menerapkan media belajar berupa majalah dengan inovasi berupa QR Code. Dimana QR Code ini menghubungkan antara media belajar majalah dengan media belajar digital berupa website game online dan video animasi pembahasan.
